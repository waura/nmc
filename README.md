# nmc

This is numerical calculation library.
This library contains the following items.
- Numeric Type
  - double
  - float
  - complext
- Numeric Container
  - vector
  - matrix
- Solver of linear equation
  - LU decomposition
  - BiCGSTAB
- Charge Simulation Method(CSM)
