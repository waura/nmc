# nmc

This is numerical analsyis library.
This library contains the following items.
- Numeric Type
  - int
  - double
  - float
  - complext
- Numeric Container
  - vector
  - matrix
- Solver of linear equation
  - LU decomposition
  - BiCGSTAB
- Charge Simulation Method (CSM)
- CSM applied Fast Multipole Method (FMCSM)
- FMCSM accelerated by CUDA
